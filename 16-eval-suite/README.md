# Lab 16: Eval Suite

Build and run a small evaluation harness for LLM outputs.

## Run

```bash
python3 16-eval-suite/eval_suite.py
python3 16-eval-suite/eval_suite.py safety
```

## What it shows

- eval cases as data
- multiple assertion types
- pass/fail summaries
- tag-based filtering
- a direct extension of the Lab 10 `eval_runner` idea
